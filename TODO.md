## 3. 아이디/비밀번호 로그인 시스템

### 설계 확정
- **닉네임 = 로그인 ID** (앱 전체 유일값, 중복 불가)
- **가입 필드**: 이름(본명·어드민만 확인) / 소속(1청~4청 드롭다운) / 닉네임 / 비밀번호
- **로그인 필드**: 닉네임 + 비밀번호
- **자동 로그인**: 로그인 성공 시 localStorage에 저장 → 같은 기기면 앱 열 때마다 자동 로그인
- **기기 변경**: localStorage 없으니 다시 닉네임+비밀번호 입력
- **비밀번호 분실**: 어드민이 직접 초기화 (문의 탭으로 요청)
- **이름 비공개 안내**: 가입 화면에서 "이름은 관리자만 확인하며 닉네임으로만 표시됩니다" 명시

### GAS `Users` 시트
```
닉네임 | 비밀번호 | 이름 | 소속 | 가입일시
```
