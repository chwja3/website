# Admin Dashboard Layout Tuning Context

## 결정

이번 변경은 admin 대시보드의 표시 순서와 카드 가독성만 수정한다. GAS 응답 구조와 집계 로직은 바꾸지 않는다.

## UI 기준

- 교구별 이번 주 참여는 관리자에게 즉시 필요한 운영 현황이므로 주차별 항목 기록보다 위에 둔다.
- 교구별 색상은 기존 앱 가입자 명단의 `badge-1청`, `badge-2청`, `badge-3청`, `badge-4청` 색상과 같은 계열을 사용한다.
- VIP와 교회학교/목양교구는 기존 badge 색상이 없어서 보조 색상만 추가한다.
- `날짜 합계`는 의미가 쪼개지지 않도록 `white-space: nowrap`으로 한 줄 유지한다.
