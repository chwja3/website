# Admin Dashboard Weekly Parish History Plan

## 목표

admin 대시보드의 `주차별 항목 기록` 아래에 `주차별 교구 참여 기록`을 추가한다.

## 구현 방향

1. GAS `dashboard` 응답에 주차별 교구 참여 요약 배열을 추가한다.
2. admin 대시보드에 주차 선택 버튼, 교구 요약 카드, 교구별 상세 목록을 추가한다.
3. 기존 `교구별 이번 주 참여` 카드 스타일과 색상 체계를 재사용한다.
4. 기존 이번 주 대시보드 수치와 항목별 기록 동작은 바꾸지 않는다.

## 검증 기준

- `dashboard` 응답에 `weeklyParishSummaries`가 포함된다.
- `주차별 항목 기록` 아래에 `주차별 교구 참여 기록` 카드가 보인다.
- 주차 버튼을 누르면 해당 주차의 교구별 참여 요약과 상세 목록이 바뀐다.
- 기존 admin HTML 스크립트 문법과 GAS 문법 검사가 통과한다.
