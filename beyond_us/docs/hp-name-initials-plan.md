# H&P 실명 초성 힌트 계획

H&P 힌트가 닉네임이나 표시명이 아니라 사용자 실명에서 나온 초성을 안정적으로 보여주게 한다.

1. `profiles`에 실명 기준 초성 필드를 추가한다.
2. 기존 사용자와 향후 가입자를 위해 이름 변경 시 초성이 자동 갱신되게 한다.
3. H&P 힌트 RPC가 `profiles.name_initials`를 우선 사용하게 한다.
4. 관리자 H&P 현황에서 사용자와 카드 작성자의 초성을 같이 확인할 수 있게 한다.

검증 기준은 H&P 힌트 요청 결과가 `이름 초성: ...` 형식으로 나오고, admin H&P 현황에서 실명 초성이 표시되는 것이다.
