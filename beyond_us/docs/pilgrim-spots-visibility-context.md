# 천로역정 스팟 표시 보정 컨텍스트

## 확인 내용

- `BBB_M3_SPOTS`에는 스팟 7개가 이미 정의되어 있다.
- 지도 이미지 파일도 `beyond_us/images/천로역정맵.webp`에 존재한다.
- 문제 가능성이 높은 지점은 `loadBBB()`의 `!bbbRes.ok` 분기다.
- 이 분기에서 기존 코드는 DEV 모드에서만 `_bbbRenderM3Spots()`를 호출했다.
- 그래서 일반 사용자에게 BBB 매칭 데이터가 아직 없으면, M3 지도는 보이는데 스팟은 비어 보일 수 있다.

## 결정

- 천로역정 M3 스팟 표시는 BBB 매칭 성공 여부와 분리한다.
- M3 탭이 열려 있거나 DEV 모드라면 빈 스팟 7개를 렌더한다.
- 오버레이 컨테이너에 `position:absolute; inset:0`을 명시해서 지도 이미지 위 레이어임을 고정한다.
