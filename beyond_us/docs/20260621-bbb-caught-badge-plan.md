# BBB 들킴 배지 오류 수정 계획

BBB 케어버디 영역의 들킴 배지가 다른 사람의 정답 상태까지 섞어서 표시되는 문제를 수정한다.

## 확인 기준

- 들킴 배지는 내가 챙기는 케어버디가 나를 시크릿버디로 맞췄을 때만 true가 된다.
- 나를 케어버디로 둔 다른 사람이 자신의 시크릿버디를 맞춘 상태는 내 들킴 배지에 영향을 주지 않는다.
- TF 추가 케어버디 구조에서는 내가 맡은 추가 케어버디가 나를 맞춘 경우만 true가 된다.
- 앱 파일은 건드리지 않고 `get_bbb_status` RPC의 `caughtByBuddy` 계산만 교정한다.

## 작업 순서

1. 현재 `caughtByBuddy` 계산 방향을 확인한다.
2. 내가 돌보는 대상 목록을 기준으로 들킴 여부를 계산하도록 SQL을 교체한다.
3. 기존 케어버디, 시크릿버디 이름 fallback과 사진 payload 반환은 유지한다.
4. 문법상 위험한 변경이 없는지 diff와 검색 검사를 실행한다.
