# Coming Soon 탭 게이트 Plan

## 목적

탭 활성화 상태가 `Coming Soon`일 때 메뉴에는 날짜 태그를 보여주되, 실제 기능 페이지로 들어가지 않고 해당 탭이 무엇인지 설명하는 단일 안내 화면을 보여준다.

## 범위

1. 앱 drawer의 `Coming Soon` 날짜 태그는 유지한다.
2. `status !== open`인 탭을 클릭하면 실제 섹션 대신 공통 안내 섹션을 표시한다.
3. B.B.B 미션은 `6/20`, 천로역정은 `6/21` 날짜와 짧은 설명을 보여준다.
4. 오픈 상태인 탭은 기존처럼 정상 기능을 보여준다.
5. 숨김 상태인 탭은 기존처럼 메뉴에서 보이지 않는다.

## 검증

- B.B.B 미션이 Coming Soon이면 일반 유저가 실제 BBB 카드, 사진 업로드, 메시지 입력을 볼 수 없어야 한다.
- 천로역정이 Coming Soon이면 지도와 업로드 UI 대신 안내 화면이 보여야 한다.
- `APP_VERSION`, `version.txt`, `app.html`, `sw.js` 버전이 동기화되어야 한다.
