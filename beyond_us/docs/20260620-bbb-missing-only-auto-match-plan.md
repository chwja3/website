# BBB 미배정 자동 매칭 보강 계획

## 목표

기존 케어버디와 시크릿버디 배정을 유지한 채, 아직 배정되지 않은 BBB 조원만 자동으로 채운다.

## 범위

- 기존 `admin_auto_assign_bbb_buddies`는 그대로 둔다.
- 새 RPC `admin_auto_fill_missing_bbb_buddies`를 추가한다.
- 일반 조/참여구분 버킷은 비어 있는 케어버디와 시크릿버디만 채운다.
- 토참/일참 단독 버킷은 전참 케어버디와 TF 시크릿버디를 배정한다.
- TF가 여러 단독 부분참여자를 케어해야 하는 경우 `bbb_extra_care_roster_links`에 추가 링크를 남긴다.
- 관리자 BBB 매칭 화면에 “미배정만 자동 매칭” 버튼을 추가한다.

## 검증

- SQL 파일에 충돌 마커가 없어야 한다.
- 관리자 HTML 인라인 스크립트 문법 검사를 통과해야 한다.
- 기존 전체 자동 매칭 버튼과 새 미배정 자동 매칭 버튼이 서로 다른 RPC를 호출해야 한다.
