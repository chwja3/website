# 2026-06-06 광범위수사와 익명 고민상담 탭 계획

## 목표
- 사용자 앱에 광범위수사 탭을 추가하고 `images/Police.webp` 포스터를 표시한다.
- 사용자 앱에 익명 고민상담 탭을 추가한다.
- 익명 고민상담은 기본적으로 본인만 조회하고, 사용자가 공개 체크를 한 경우에만 이름 없이 모든 유저가 내용만 볼 수 있게 한다.

## 범위
- 앱 드로어 메뉴와 섹션 전환에 `investigation`, `counseling` 탭을 추가한다.
- Supabase `tab_settings` 기본 항목과 앱 bootstrap JSON에 두 탭 키를 추가한다.
- Supabase에 익명 고민상담 전용 테이블과 사용자 RPC를 추가한다.
- 관리자 페이지에는 고민 내용을 보여주는 화면을 만들지 않는다.

## 검증
- `app.html`과 `app.js` 문법 검사를 실행한다.
- 새 SQL 마이그레이션에 필요한 함수와 권한 구문이 들어갔는지 정적 검사한다.
- `git diff --check`로 공백 오류를 확인한다.
