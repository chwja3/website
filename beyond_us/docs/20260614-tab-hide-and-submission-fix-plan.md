# 2026-06-14 탭 숨김과 등록 실패 보강 계획

## 목표

광범위수사는 설명 문구가 확정될 때까지 사용자 앱에서 숨김 상태로 둔다. 목사님께 무물과 별빛 우편함은 사용자가 등록 버튼을 눌렀을 때 Supabase RPC 시그니처 불일치나 최신 SQL 일부 누락 때문에 실패하지 않도록 보강한다.

## 작업 범위

1. 광범위수사 탭은 DB 기본값을 `enabled=false`, `status=closed`로 고정한다.
2. 나중에 광범위수사를 닫힘 상태로 노출할 때 햄버거 메뉴에 `6/20` 날짜 배지가 나오도록 앱 날짜 매핑을 추가한다.
3. 목사님께 무물의 사용자 등록, 수정, 삭제 RPC를 최신 형태로 다시 정의한다.
4. 별빛 우편함의 사용자 등록, 수정 RPC에 현재 앱이 보내는 `p_is_anonymous` 시그니처를 다시 정의한다.
5. PostgREST schema cache reload 알림을 포함해 SQL 실행 직후 반영되도록 한다.

## 검증 기준

- `app.js` 문법 검사가 통과한다.
- 새 SQL 파일에 `notify pgrst, 'reload schema'`가 포함된다.
- 광범위수사는 새 SQL 실행 후 기본적으로 사용자 드로어에서 보이지 않는다.
- 목사님께 무물과 별빛 우편함 등록 함수는 현재 앱 호출 인자명과 일치한다.
