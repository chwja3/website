## 확인이 필요한 핵심 질문

- BBB 매칭은 조 편성 이후 누가 어떤 시트에 조·티어 정보를 입력하는가.
- 크라임씬 맵은 단순 이미지 열람인지, 위치별 단서 클릭형 게임인지, 진행 상태 저장이 필요한지.
- 유형검사는 문항 수, 결과 유형, 점수 계산식, 결과 저장 여부가 무엇인지.
- 개인별 현장 미션 2개는 랜덤 배정인지, 조별·개인별 지정인지, admin 확인이 필요한지.
- 추첨권 실물 번호는 앱 번호와 어떻게 매칭해 배부/투입하는지.
- 레어 엔 카드 최종 이미지/명칭/사운드는 현재 자산을 유지할지, 새 자산을 제작할지.
- 미보유 엔 카드 지급권 수동 지급·회수 기능을 나중에 admin 보정 도구로 둘지.
- 레어카드 획득 연출은 어느 순간에 보여줄지, 다시보기를 둘지.
- H&P에서 코드에 남아 있는 하드코딩 항목은 무엇이며, 어느 시트나 admin 설정으로 옮길지.
- Q.T. 본문은 날짜별 자동 노출인지, 운영자 수동 공개인지.
