# 향후 기능 백로그 Context

## 입력

2026-05-13 기준 사용자가 다음 항목을 향후 작업 목록으로 제시했다.

1. 현장미션 MISSION 3 위치 정리.
2. BBB 매칭을 프로그램팀 수련회 조 편성 이후 조별·티어별로 재설계.
3. 히든 카드 미션 수정.
4. 수련회 말씀 묵상 Q.T. 페이지.
5. 크라임씬 맵 기반 앱 안 광범위수사.
6. 내 안의 빛 유형검사 문항 구현.
7. 구성 탭 레이아웃.
8. 레어카드 뽑기와 더 화려한 카드팩 이펙트.
9. 현장 미션 개인별 두 개.
10. 추첨권.

## 현재 판단

- 1, 4, 7번은 사용자가 완료로 분류했다.
- 나머지는 아직 정책과 구체 UX가 부족해서 곧바로 구현하기 어렵다.
- 특히 2, 3, 5, 6, 9, 10은 Google Sheet 구조와 admin 운영 방식까지 같이 정해야 한다.
- 현장 미션 보상은 BBB MISSION 1, 2, 3의 특별 카드팩 지급으로 구현됐다.
- 특별 카드팩은 내부 계수만 일반 카드팩과 분리하고, 유저 앱에서는 단일 “카드팩” 버튼으로 통합한다.
- 특별 카드팩은 개봉 시점의 최신 Collection 기준으로 미보유 일반 카드가 있으면 미보유 카드 확정, 9종을 모두 보유했으면 일반 카드팩처럼 1-9번 랜덤 지급한다.
- 특별 카드팩 후보는 일반 카드 9종만 포함하고, 히든/레어카드는 무조건 제외한다.
- `🎫` 배지는 일반+특별 전체 카드팩 수, `💜` 배지는 그중 특별 카드팩 수를 표시한다.

## 확인이 필요한 핵심 질문

- BBB 매칭은 조 편성 이후 누가 어떤 시트에 조·티어 정보를 입력하는가.
- 히든 카드 미션은 어떤 행동을 하면 어떤 카드나 티켓을 주는가.
- 크라임씬 맵은 단순 이미지 열람인지, 위치별 단서 클릭형 게임인지, 진행 상태 저장이 필요한지.
- 유형검사는 문항 수, 결과 유형, 점수 계산식, 결과 저장 여부가 무엇인지.
- 레어카드는 기존 9종 카드와 같은 컬렉션인지, 별도 등급인지, 확률과 중복 정책은 무엇인지.
- 개인별 현장 미션 2개는 랜덤 배정인지, 조별·개인별 지정인지, admin 확인이 필요한지.
- 추첨권은 뽑기권과 별도인지, 카드 수집 개수와 연동되는지, admin에서 추첨 관리가 필요한지.
- 특별 카드팩 수동 지급·회수 기능을 나중에 admin 보정 도구로 둘지.
