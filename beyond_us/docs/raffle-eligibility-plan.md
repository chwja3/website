# 추첨권 대상 교구 제한 Plan

## 목표

추첨권은 1청, 2청, 3청, 4청, VIP에게만 보이게 하고 발급한다. 목양교구와 교회학교 사용자는 추첨권 번호를 새로 받지 않고, 앱에서 추첨권 화면도 보지 않는다.

## 작업 범위

- 서버에서 추첨권 대상 교구 판정 함수를 추가한다.
- 가입, 카드 보유 조건 backfill, 카드 뽑기 이후 자동 발급이 모두 대상자에게만 동작하게 한다.
- 추첨권 통계의 전체 수량과 참여자 수에서도 비대상자를 제외한다.
- 비대상 사용자의 앱 컬렉션 헤더에서 추첨권 배지를 숨긴다.
- 기존 목양교구/교회학교 추첨권 번호는 회수 함수로 `revoked_ineligible_parish` 상태로 표시한다.
- admin의 수련회 참석 탭을 앱 가입자 탭으로 바꾸고, 가입자별 추첨권 초기화 버튼을 추가한다.
- 가입자별 추첨권 초기화는 해당 유저의 활성 추첨권만 `revoked_admin_reset` 상태로 회수한다.

## 제외 범위

- 기존 번호를 물리적으로 삭제하거나 번호를 재사용하지 않는다.
- admin 참석자 체크 UI는 앱 가입자 탭 안에 유지한다. 현재 참석 체크는 추첨권에 반영되지 않는다.
