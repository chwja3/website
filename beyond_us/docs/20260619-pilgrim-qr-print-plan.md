# 천로역정 QR 출력 UI 계획

## 목표

관리자가 천로역정 7개 장소별 QR을 바로 확인하고 인쇄할 수 있게 한다.

## 범위

- admin `BBB/천로역정` 탭의 천로역정 QR 영역을 URL 복사 목록에서 QR 출력 카드 목록으로 바꾼다.
- 각 카드에는 스팟 번호, 장소명, QR 이미지, URL 복사, PNG 열기 버튼을 둔다.
- 전체 QR을 별도 인쇄창으로 열어 A4 출력하기 쉽게 만든다.

## 검증

- admin inline script가 문법 오류 없이 파싱되는지 확인한다.
- 충돌 마커와 whitespace 오류를 확인한다.
