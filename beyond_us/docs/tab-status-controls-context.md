# 탭 상태 표시 관리 컨텍스트

## 결정

- `enabled`와 `status`를 분리한다.
- `enabled=false`는 유저 앱 메뉴에서 완전히 숨기는 상태다.
- `status=open`은 초록색, `status=closed`는 빨간색으로 표시한다.
- 빨간색 상태라도 `enabled=true`면 사용자가 탭에 들어갈 수 있다. 실제 잠금/Coming Soon 콘텐츠는 각 탭 내부 로직이 담당한다.
- 채팅방은 기존 앱 섹션이 남아 있으므로 `TabSettings`에 포함하되 기본값은 숨김으로 둔다.
- 2026-05-16. `TabSettings` 기본 행을 공지사항, 사전미션, Hold & Pray, Q.T., B.B.B, 천로역정, 카드 컬렉션, QnA, 개발자 문의, 채팅방, 미보유 카드 지급권까지 확장했다.
- admin `탭 활성화` 화면은 GAS가 내려주는 `items` 배열을 렌더링하며, 각 행에서 표시 여부와 초록/빨강 상태를 저장한다.
- 유저 앱 드로어는 표시 중인 탭 옆에 초록/빨강 상태 점을 붙인다. 프론트 캐시 버전은 `20260516a`다.
