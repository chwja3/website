# Q.T. 말씀 묵상 탭 추가 컨텍스트

## 목표

- 수련회 현장에서 사용할 말씀 묵상 Q.T. 페이지의 진입점을 미리 만든다.
- 실제 콘텐츠는 아직 확정 전이므로 앱 페이지는 `준비중` 레이아웃만 둔다.
- 관리자 `탭 활성화` 화면에서 Q.T. 탭 노출 여부를 제어할 수 있게 한다.

## 구현 결정

- 탭 키는 영문 key 원칙에 맞춰 `qt`로 둔다.
- 앱 표시 라벨은 `Q.T. 말씀 묵상`으로 둔다.
- Q.T. 메뉴는 사용자 주요 동선에 가까운 기능이므로 드로어 상단 주요 메뉴에 배치한다.
- 기존 `chat` 키는 하위 호환을 위해 GAS에는 유지하되, 관리자 UI에는 다시 노출하지 않는다.
- 준비중 화면은 별도 입력/저장 기능 없이 안내 카드만 둔다.
