# 하진 프로필 보정과 H&P 정답 정책 핫픽스 계획

## 목표

앱 가입자 `하진`을 실제 조 명단 이름 `유하진`으로 보정하고, H&P 정답 판정을 모든 비익명 카드에서 일관되게 처리한다.

## 범위

- 4조 조 명단의 `유하진` row를 앱 가입 프로필에 매칭한다.
- 프로필 이름이 아직 `하진`이면 `유하진`으로 변경한다.
- H&P 제출, 조회, 재계산에서 비어 있지 않은 입력을 모두 정답으로 본다.
- 익명 카드는 기존처럼 입력과 정답 처리를 하지 않는다.
- 기존 카드팩 지급 함수는 유지하고, 재계산 후 누락 보상을 자동 지급한다.

## 검증

- SQL 파일에 충돌 마커가 없어야 한다.
- H&P 관련 함수명이 예상대로 포함되어야 한다.
- 프로필 보정 SQL이 중복 대상일 때는 조용히 잘못 업데이트하지 않아야 한다.
