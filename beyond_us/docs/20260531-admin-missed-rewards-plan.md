# 2026-05-31 관리자 누락 보상 지급 계획

## 목표
- 앱 가입자 탭에서 유저별 누락 보상을 확인하고 지급할 수 있게 한다.
- 관리자가 임의로 수량을 넣는 방식이 아니라, 서버가 조건 충족 여부와 기존 지급 여부를 판단한다.
- 이미 받은 카드 뽑기권이나 추첨권은 비활성 상태로 보이게 한다.

## 1차 대상
- 사전미션 주차별 카드 뽑기권.
- H&P w3, w6 정답 카드 뽑기권.
- 추첨권 자동 조건인 앱 가입, 카드 3종, 카드 5종, 카드 10종.

## 검증
- SQL RPC가 조건 미충족과 중복 지급을 막는지 정적 확인한다.
- 관리자 페이지 스크립트가 파싱되는지 확인한다.
- `git diff --check`로 공백 오류를 확인한다.
