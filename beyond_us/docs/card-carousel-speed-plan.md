# 카드팩 캐러셀 로딩 단축 계획

## 목표

카드 뽑기 버튼을 누른 뒤 카드팩 캐러셀이 뜨기까지의 체감 지연을 줄인다.

## 범위

1. 카드팩 선택 화면에 필요한 팩 이미지와 카드 뒷면 이미지를 먼저 로드한다.
2. 오버레이 첫 화면을 띄우기 전 실행되는 사운드 프리로드와 이펙트 DOM 생성을 뒤로 미룬다.
3. 캐러셀 등장 애니메이션을 조금 짧게 조정한다.
4. 프론트 변경에 맞춰 `APP_VERSION`, `version.txt`, `sw.js`, `app.html` 쿼리 버전을 동기화한다.

## 검증 기준

- 카드 뽑기 버튼 클릭 직후 캐러셀 레이어가 이전보다 빠르게 보인다.
- 카드팩 이미지가 빈 상태로 늦게 나타나지 않는다.
- 카드 선택, 팩 찢기, 카드 공개 흐름은 유지된다.
- `node --check beyond_us/app.js`와 `git diff --check`를 통과한다.
