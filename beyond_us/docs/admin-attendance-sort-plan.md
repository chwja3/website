# Admin 앱 가입자 정렬 계획

admin의 `앱 가입자` 목록을 운영자가 명단처럼 읽기 쉽게 교구순, 이름순으로 정렬한다.

1. Supabase에서 가입자 목록을 내려주는 전용 RPC를 추가한다.
2. 서버에서 교구 고정 순서와 이름순 정렬을 적용한 뒤 limit을 적용한다.
3. admin 프론트의 앱 가입자 탭은 기존 `admin_dispatch` action 대신 새 RPC를 호출한다.
4. 캐시 버전을 갱신하고 문법 검사를 통과시킨다.

교구 순서.

`1청`, `2청`, `3청`, `4청`, `VIP`, `교회학교/목양교구`, 기타.
