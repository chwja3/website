# 2026-06-12 관리자 카드 뽑기권 누락 보상 복원 계획

## 목표

앱 가입자 탭의 `뽑기권/추첨권 현황` 카드에 예전 누락 보상 기능을 카드 뽑기권에 한해서 다시 표시하고 지급할 수 있게 한다.

## 성공 기준

- 현황 카드에서 카드 뽑기권 누락 보상 목록이 보인다.
- 목록에는 사전미션 주차 보상과 H&P w3/w6 카드 뽑기권 보상만 포함된다.
- 추첨권 조건 보상은 목록에 다시 넣지 않는다.
- 지급 버튼은 카드 뽑기권만 지급한다.
- 기존 추첨권 추가와 회수 기능은 유지된다.

## 구현 범위

- `admin_get_user_ticket_status`가 카드 뽑기권 보상 목록을 함께 반환하도록 갱신한다.
- `admin_issue_user_missed_reward`를 카드 뽑기권 전용 지급 함수로 추가한다.
- admin UI의 조건형 보상 제목을 카드 뽑기권 누락 보상으로 바꾼다.

## 검증

- admin HTML inline script 문법을 확인한다.
- 예전 추첨권 조건 보상이 SQL에 포함되지 않았는지 확인한다.
- 충돌 마커와 공백 오류를 확인한다.
