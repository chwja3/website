# 관리자 운영 상태 화면 개편 계획

목표.

1. 개발자 문의 탭에서 H&P 힌트 요청 영역을 제거한다.
2. 문의는 미답변과 답변 완료를 분리하고, 각 문의 내용은 기본 닫힘 상태로 여닫을 수 있게 한다.
3. H&P 운영 조회 탭을 추가해 유저별 기도제목, 맞힌 카드 수, 정답 여부를 확인한다.
4. 기존 BBB 사진 승인 탭을 BBB/천로역정 탭으로 확장해 BBB 매칭, Mission 1/2 승인, 천로역정 스팟과 성공 여부를 한 화면에서 본다.
5. Supabase 기준 운영 조회 RPC를 추가해 프론트에서 여러 테이블을 직접 조합하지 않는다.
