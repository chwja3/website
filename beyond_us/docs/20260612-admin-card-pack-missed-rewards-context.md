# 2026-06-12 관리자 카드 뽑기권 누락 보상 복원 컨텍스트

## 배경

- `뽑기권/추첨권 현황`으로 바꾸는 과정에서 예전 `누락 보상` 목록 의존성을 제거했다.
- 사용자가 원래 있던 누락 보상을 카드 뽑기권에 한해서는 그대로 유지하길 요청했다.

## 결정

- 추첨권 조건형 보상은 복원하지 않는다.
- 사전미션 주차 기준 카드 뽑기권과 H&P w3/w6 카드 뽑기권만 복원한다.
- UI는 기존 현황 카드 안에 `카드 뽑기권 누락 보상` 섹션으로 보여준다.
- 지급 함수 이름은 기존 admin 코드 경로를 유지하기 위해 `admin_issue_user_missed_reward`를 사용하되, 내부는 카드 뽑기권만 처리한다.

## 주의점

- `admin_get_user_ticket_status`가 현황과 카드 뽑기권 보상 목록을 함께 반환한다.
- `admin_issue_user_missed_reward`는 `raffle_*` 보상 key를 처리하지 않는다.
- 2026-06-12 후속 결정으로 카드 뽑기권 보상은 조건 미충족이어도 미지급이면 지급 가능하게 바뀌었다.
- 수동 지급이 조건 미충족 상태에서 이루어진 경우 이벤트 payload의 `adminOverride`가 true가 된다.
