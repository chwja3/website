# 2026-05-31 관리자 카드 표시명 변경 계획

## 목표
- 관리자 화면의 카드별 뽑힌 횟수, 유저별 실물 수령, 카드 이벤트 생성에서 카드 이름을 성령의 열매 이름 대신 실제 카드 캐릭터 이름으로 표시한다.
- 사용자 앱과 Supabase의 `cards.name` 데이터는 유지해서 기존 이벤트와 집계 구조를 흔들지 않는다.

## 표시명
- 1 사모, 2 라라, 3 달래, 4 오참, 5 더무, 6 네해, 7 고, 8 엔, 9 버터, 10 BetWeEn.

## 검증
- 관리자 HTML의 카드 표시 함수가 모든 주요 관리자 카드 화면에 적용됐는지 확인한다.
- 변경 대상은 관리자 단일 HTML이라 앱 PWA 버전 변경은 하지 않는다.
- HTML/JS 정적 검사를 실행한다.
