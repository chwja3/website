# 2026-05-31 관리자 누락 보상 지급 컨텍스트

- 운영 중 H&P처럼 조건은 성공했지만 보상 지급이 누락되는 사례가 있었다.
- 사용자가 원하는 것은 무제한 수동 발급이 아니라, 놓친 과거 미션 보상을 조건 기반으로 수동 회수/보정하는 운영 도구다.
- 첫 범위는 기존 데이터로 판정 가능한 사전미션, H&P, 추첨권 자동 조건으로 제한한다.
- 앱 가입자 탭에서는 유저별 “누락 보상” 버튼으로 조회하고, 서버가 `available=true`로 판정한 항목만 지급 버튼을 활성화한다.
- 이미 지급된 보상은 해당 보상 이벤트나 활성 추첨권 레코드를 기준으로 비활성화한다.
