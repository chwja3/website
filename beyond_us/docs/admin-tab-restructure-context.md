# Admin 탭 정리 컨텍스트

## 결정 사항

- 기존 `tab-checkin` 패널이 사실상 대시보드 역할을 하고 있었지만 드로어에 메뉴가 없어 접근성이 떨어졌다.
- 운영자 공통 메뉴는 `대시보드`, `공지`, `실물 카드 수령`, `개발자 문의`, `유저 비밀번호 초기화` 순서로 둔다.
- 개발자 중심 메뉴는 `탭 활성화`, `B.B.B 미션`, `Events 관리`, `시스템 상태`, `주차·미션 설정` 순서로 둔다.
- `실물 카드 수령`은 유지한다. `getCardStats()`가 현재 `Events`, `CardReceived`, `Trades`를 조합하므로 예전 `CardDraws` 기준 화면이 아니다.
- 별도 `카드 집계` 탭은 제거한다. 이 탭은 `Collection` 캐시 재계산 안내와 버튼만 담고 있고, `Events 관리`의 `Events 기준 재계산`과 역할이 겹친다.

## 구현 메모

- `tab-collection` 패널과 `runRebuildCollection()` UI 진입점은 제거한다.
- `adminRebuildCollection` GAS 액션 자체는 하위 호환과 수동 복구용으로 남긴다.
- 관리자 탭 이름은 사용자 앱과 맞춰 `현장미션 (BBB)` 대신 `B.B.B 미션`으로 정리한다.
- DEV 정적 버전은 `20260512x`로 갱신한다.
