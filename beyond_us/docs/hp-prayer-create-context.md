# H&P 새 기도제목 추가 컨텍스트

2026-05-20. 운영 중 새로 들어오는 H&P 기도제목을 Admin H&P 기도제목 매칭 탭에서 직접 입력할 수 있게 한다.

기존 `admin_upsert_hold_pray_entry`는 특정 유저와 주차 기준으로 기도제목을 upsert한다. 이번 기능은 새 원천 기도제목을 추가하는 목적이므로 기존 유저의 기도제목을 덮어쓰지 않는 별도 insert RPC가 필요하다.

새 기도제목은 특정 주차에 묶지 않고 `week_key = null`로 저장한다. H&P 본문과 사람 매칭은 주차별 상태가 아니라 전체 기도제목 원천 데이터로 보기 때문이다.

작성자 매칭은 기존 매칭 탭 정책을 따른다. 닉네임이 있으면 닉네임을 우선하고, 없으면 실명 단일 매칭으로 연결한다. 동명이인 또는 미가입자는 `owner_name_input`만 저장해 기존 매칭 탭에서 후속 처리한다.

새 기도제목을 사용자 화면에 바로 노출할지 선택할 수 있게 한다. 노출된 새 기도제목은 H&P 카드 풀에 포함되므로, 이미 사용자가 H&P를 확인한 뒤 추가하면 카드 구성이 바뀔 수 있다.
