# 2026-06-06 익명 고민상담 관리자 답변 계획

## 목표
- admin 페이지에서 익명 고민상담에 답변을 달 수 있게 한다.
- 작성자 닉네임, 이름, profile_id는 admin 응답에 반환하지 않아 익명을 유지한다.
- 사용자는 본인이 남긴 고민과 공개된 익명 고민에서 관리자 답변을 확인할 수 있게 한다.

## 구현 방향
- `anonymous_counseling_entries`에 답변 본문, 답변자, 답변 시간을 저장하는 컬럼을 추가한다.
- 관리자 전용 RPC는 고민 내용과 공개 여부, 작성 시간, 답변만 반환한다.
- 사용자 조회 RPC는 본인 고민과 공개 고민에 답변 정보를 함께 내려준다.
- 사용자가 고민 본문을 수정하면 기존 답변은 해당 본문에 대한 답변이 아니게 될 수 있으므로 답변을 초기화한다.
