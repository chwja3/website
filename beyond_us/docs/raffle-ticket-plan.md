# 추첨권 구현 계획

## 목표

카드 컬렉션 탭에서 사용자가 본인의 추첨권 개수를 바로 확인하고, 추첨권 배지를 누르면 전체 추첨권과 내 추첨권이 시각적으로 강조된 추첨통 모달을 볼 수 있게 한다.

## 구현 범위

1. GAS `userStatus` 응답에 `raffle` 객체를 추가한다.
2. 추첨권 계산은 현재 컬렉션 도움말에 적힌 카드 종류 보상 정책을 기준으로 한다.
3. 컬렉션 상단 문구 옆에 추첨권 배지를 추가한다.
4. 배지를 누르면 아크릴 추첨통 모달을 열고 전체 추첨권, 내 추첨권, 통 채움 정도를 표시한다.
5. 프론트 변경에 맞춰 `APP_VERSION`, `version.txt`, `sw.js` 캐시명을 동기화한다.

## 검증 기준

- `userStatus` JSON에 `raffle.myTickets`, `raffle.totalTickets`, `raffle.fillPercent`가 포함된다.
- 컬렉션 탭에서 `n/10종 보유` 옆에 추첨권 배지가 보인다.
- 추첨권 배지를 누르면 모달이 열리고 닫힌다.
- 전체 추첨권이 1000장 이상이면 통 채움이 90%로 고정된다.
- 내 추첨권 수만큼 황금색 추첨권이 더미 위에 표시된다.
