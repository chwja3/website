# 천로역정 개인별 2스팟 미션 계획

## 목표

천로역정 7개 스팟 중 유저별로 고정 랜덤 2개를 미션 스팟으로 지정하고, 두 스팟 인증을 완료하면 레어 카드를 지급한다.

## 설계

- 스팟 배정은 유저 ID를 seed로 하는 고정 랜덤 계산으로 처리한다.
- 별도 시트 쓰기 없이 서버에서 매번 같은 2개 스팟을 계산한다.
- 서버는 배정되지 않은 M3 스팟 업로드를 거부한다.
- 프론트는 배정된 미완료 스팟을 빨간 원, 완료 스팟을 초록 원, 비배정 스팟을 흰 원으로 표시한다.
- 레어 카드 중복 지급은 기존 `bbb_m3` 보상 이벤트 기준을 유지한다.

## 검증

- `node --check beyond_us/Apps_Script`
- `node --check beyond_us/app.js`
- 버전 문자열 동기화 확인.
- diff 검사 실행.
