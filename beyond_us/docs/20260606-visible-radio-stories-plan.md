# 2026-06-06 보이는 라디오 사연 계획

## 목표
- 사용자 앱에 `보이는 라디오` 탭을 추가한다.
- 사용자는 익명 사연을 올리고 본인이 올린 사연만 확인할 수 있게 한다.
- admin 페이지에서는 모든 사연을 작성자 정보 없이 모아볼 수 있게 한다.

## 구현 방향
- Supabase에 `visible_radio_stories` 테이블과 사용자 RPC를 추가한다.
- 사용자 RPC는 로그인 유저의 사연만 조회, 작성, 수정, 삭제한다.
- 관리자 RPC는 모든 사연을 조회하되 profile_id, 닉네임, 이름을 반환하지 않는다.
- 탭 활성화는 기존 `tab_settings` 흐름에 `visible_radio` key를 추가해서 관리한다.
