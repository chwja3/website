# 사진첩 업로드 400 오류 수정 계획

## 목표

사진첩에서 고른 B.B.B. MISSION 1, 2 사진이 Supabase Storage 업로드 중 400 오류를 내지 않도록 한다.

## 범위

- Storage object 경로의 유저 폴더명을 ASCII 해시 기반으로 안정화한다.
- Storage 업로드 실패 메시지를 JSON 응답까지 읽어 더 정확하게 표시한다.
- B.B.B. 사진 input은 같은 파일을 다시 선택할 수 있게 처리한다.
- 천로역정 QR 인증 흐름은 변경하지 않는다.

## 검증

- `app.js` 문법 검사를 통과한다.
- 앱 HTML 인라인 스크립트 검사를 통과한다.
- 앱 버전과 서비스워커 캐시 버전을 함께 갱신한다.
