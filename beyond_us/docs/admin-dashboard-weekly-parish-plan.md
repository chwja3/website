# Admin Dashboard Weekly And Parish Plan

## 목표

admin 대시보드에서 이번 주 상태를 먼저 보여주고, 아래에서 1~6주차 항목별 기록과 이번 주 교구별 참여 현황을 빠르게 확인할 수 있게 한다.

## 구현 방향

1. GAS `getDashboardData()`가 모든 `mission.submitted` 이벤트를 한 번 읽고 active 유저만 대상으로 주차별 요약을 계산한다.
2. 주차별 요약에는 항목별 체크 수, 주차 참여자 수, 제출 이벤트 수, 누적 실천 수를 포함한다.
3. 이번 주 교구별 요약에는 교구 그룹별 참여자 수, 총점, 참여 날짜 수, 유저별 주차 점수와 참여 날짜 수를 포함한다.
4. admin 대시보드는 기존 이번 주 항목 테이블을 유지하되, 아래에 주차별 버튼/테이블과 교구별 summary/detail 영역을 추가한다.

## 검증 기준

- 대시보드 최상단 참여자 수는 이번 주 active 유저 기준이다.
- 1~6주차 탭을 눌렀을 때 해당 주차의 항목별 기록이 바뀐다.
- 교구별 목록은 1교구, 2교구, 3교구, 4교구, VIP, 교회학교/목양교구 그룹으로 묶인다.
- 교구 안 유저 목록은 주차 참여 점수, 참여 날짜 수 내림차순으로 정렬된다.
