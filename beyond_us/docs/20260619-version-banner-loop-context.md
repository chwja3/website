# 새 버전 배너 반복 수정 컨텍스트

2026-06-19. 사용자가 새로고침을 눌러도 "새 버전이 있어요. 새로고침해주세요." 배너가 계속 뜬다고 했다. `app.js`의 `APP_VERSION`과 `app.html`의 script query는 `20260619a`였지만, `version.txt`와 `sw.js` 캐시 이름은 `20260618b`였다.

기존 버전 체크는 원격 버전과 로컬 버전이 다르기만 하면 새 버전으로 판단했다. 이 때문에 `version.txt`가 앱보다 뒤처진 상태에서도 배너가 반복 표시됐다. `version.txt`와 service worker 캐시를 현재 버전으로 맞추고, `YYYYMMDDa` 형태의 버전은 원격이 로컬보다 클 때만 배너를 띄우도록 보강했다.
