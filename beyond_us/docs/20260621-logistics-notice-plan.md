# 숙소 안내문 상단 노출 계획

숙소/차량 배정표 탭에서 전체 숙소 이미지보다 먼저 퇴실, 셔틀, 주일 체크인 안내문을 보여준다.

## 확인 기준

- 숙소/차량 탭 진입 시 안내문이 이미지보다 먼저 보인다.
- 기존 숙소 배정 이미지는 안내문 아래에 그대로 표시된다.
- 앱 캐시 버전이 함께 갱신되어 사용자가 새 화면을 받을 수 있다.

## 작업 순서

1. 숙소 이미지 렌더링 함수 위치를 확인한다.
2. 안내문 HTML을 별도 함수로 분리해 이미지 위에 삽입한다.
3. `APP_VERSION`, `version.txt`, service worker cache 이름을 함께 갱신한다.
4. 문법 검사와 diff 검사를 실행한다.
