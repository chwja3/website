# 2026-06-12 관리자 카드 뽑기권 미지급 기준 지급 계획

## 목표

앱 가입자 탭의 카드 뽑기권 누락 보상을 조건 달성 여부와 무관하게, 해당 보상이 이미 지급된 적이 없으면 지급할 수 있게 한다.

## 성공 기준

- 사전미션 주차 카드 뽑기권은 주차 점수 조건을 달성하지 않아도 미지급이면 버튼이 활성화된다.
- H&P w3/w6 카드 뽑기권은 정답 기록이 없어도 미지급이면 버튼이 활성화된다.
- 이미 지급된 같은 보상은 계속 비활성화된다.
- 추첨권 추가와 회수 정책은 바꾸지 않는다.

## 구현 범위

- `admin_get_user_ticket_status`의 카드 뽑기권 보상 `available` 기준을 `not claimed`으로 바꾼다.
- `admin_issue_user_missed_reward`의 조건 미충족 차단을 제거한다.
- H&P 수동 지급은 기존 자동 지급 보조 함수 대신 직접 `ticket.granted` 이벤트와 `user_inventory`를 갱신한다.

## 검증

- SQL에서 `condition_not_met` 차단이 남아 있지 않은지 확인한다.
- admin HTML inline script 문법을 확인한다.
- 충돌 마커와 공백 오류를 확인한다.
