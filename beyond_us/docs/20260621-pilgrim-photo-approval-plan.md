# 2026-06-21 천로역정 사진 승인 전환 계획

## 목표

천로역정은 유저별 랜덤 2스팟 배정을 유지하되, 완료 인증 방식을 QR 즉시 완료에서 BBB Mission 1/2와 같은 사진 업로드 후 관리자 승인 방식으로 전환한다.

## 성공 기준

1. 유저는 배정된 빨간 스팟을 눌러 인증 사진을 업로드할 수 있다.
2. 서버는 `m3_*` 또는 `pilgrim` 업로드를 `qr_required`로 막지 않고 `mission_photo_submissions`에 `pending`으로 저장한다.
3. 배정되지 않은 스팟 업로드는 `not_assigned_spot`으로 거절한다.
4. admin 승인 시 기존 BBB 승인 흐름처럼 해당 스팟이 완료로 집계되고, 2스팟 모두 승인되면 천로역정 보상이 지급된다.
5. QR 스캐너 UI는 유저 화면에서 더 이상 열리지 않는다.
