# 관리자 천로역정 QR PNG 저장 수정 계획

관리자 개발자 탭의 천로역정 QR `PNG 열기` 버튼이 팝업 또는 새 창 환경에서 계속 실패해, 브라우저 다운로드 방식의 `PNG 저장`으로 바꾼다.

1. 기존 팝업 기반 `PNG 열기` 경로를 제거한다.
2. QR 이미지를 canvas에 그려 PNG Blob으로 변환한다.
3. PNG 상단에 `천로역정 스팟 n / 이름 / 인증 QR` 제목과 안내 문구를 넣는다.
4. 버튼은 `PNG 저장`으로 표시하고, 클릭 시 파일을 바로 다운로드한다.
5. 문법 검증 후 main에 반영한다.
