# 천로역정 스팟 표시 보정 계획

## 목표

천로역정 지도에 정의된 7개 스팟이 사용자 화면에서 안정적으로 표시되도록 한다.

## 범위

- 프론트엔드의 천로역정 지도 오버레이 렌더링만 수정한다.
- BBB 매칭 데이터가 아직 없거나 `getBBB`가 `ok:false`를 반환해도, M3 영역이 표시되는 상태라면 빈 스팟 7개를 렌더링한다.
- 지도 이미지 위 스팟 레이어의 위치 기준과 클릭 가능 영역을 명시한다.

## 검증

- `BBB_M3_SPOTS` 7개 정의 유지 확인.
- `node --check beyond_us/app.js` 통과.
- 버전 문자열 동기화 확인.
