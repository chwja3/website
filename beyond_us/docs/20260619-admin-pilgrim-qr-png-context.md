# 관리자 천로역정 QR PNG 저장 수정 컨텍스트

2026-06-19. 사용자는 main 관리자 개발자 탭에서 천로역정 QR `PNG 열기` 버튼이 여전히 정상 동작하지 않는다고 했다. 이전 구현은 새 창을 열고 canvas data URL을 그 창에 표시하는 구조였는데, 팝업 차단이나 새 창 렌더링 문제를 계속 피하지 못했다.

결정. 이제 QR PNG는 열지 않고 저장한다. 버튼 이름도 `PNG 저장`으로 바꾸고, 클릭 시 canvas를 PNG Blob으로 만든 뒤 `<a download>`를 통해 직접 파일 다운로드를 실행한다. QR 이미지는 기존 `api.qrserver.com` URL을 canvas에 그리는 방식을 유지하고, PNG에는 `천로역정 스팟 n / 이름 / 인증 QR` 제목과 `해당 장소에 붙여두는 천로역정 인증 QR입니다.` 안내 문구를 포함한다.
