# 관리자 천로역정 QR 개별 인쇄 전환 컨텍스트

2026-06-19. 사용자는 전체 인쇄 버튼은 정상 동작하지만 개별 PNG 저장 버튼은 계속 반응하지 않는다고 했다. 브라우저 다운로드 정책이나 파일 저장 API보다, 이미 검증된 전체 인쇄 흐름을 개별 QR에도 쓰는 편이 운영상 안정적이다.

결정. QR 카드의 보조 버튼은 `PNG 저장`이 아니라 `인쇄`로 바꾼다. 전체 인쇄 함수에서 쓰던 새 창 생성, QR 이미지 로딩 대기, `print()` 호출 흐름을 `printPilgrimQrRows`로 분리하고, 전체 인쇄는 전체 rows를, 개별 인쇄는 단일 row를 넘기게 한다. PNG 저장 전용 canvas/download 함수들은 더 이상 쓰지 않으므로 제거한다.
