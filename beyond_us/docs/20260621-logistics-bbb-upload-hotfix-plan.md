# 숙소 안내 이미지 교체와 BBB 업로드 보강 계획

사용자가 제공한 2026년 6월 21일 기준 숙소 배정 안내 이미지 2장을 숙소/차량 탭의 공통 안내 이미지로 교체한다.

BBB Mission 1, 2 사진 업로드는 갤러리 선택 후 이미지 압축 단계가 멈추거나 실패해도 사용자에게 원인을 보여주도록 보강한다.

## 작업 범위

- 기존 숙소 안내 이미지 1, 2쪽을 새 이미지로 교체한다.
- 숙소 탭에서 기존 3쪽 이미지는 더 이상 렌더링하지 않는다.
- 숙소 안내 문구의 기준일을 2026년 6월 21일로 바꾼다.
- BBB 사진 압축 함수에 타임아웃과 명확한 오류 처리를 추가한다.
- Mission 1, 2 업로드 핸들러가 같은 안정화 로직을 사용하게 한다.
