# 2026-06-12 관리자 카드 뽑기권 직접 지급 계획

## 목표

운영자가 앱 가입자 탭에서 이름 또는 닉네임으로 유저를 찾은 뒤 카드 뽑기권 1장을 바로 지급할 수 있게 한다.

## 성공 기준

- 앱 가입자 목록의 각 유저 행에 카드 뽑기권 직접 지급 버튼이 보인다.
- 버튼 클릭 시 운영자 권한으로 해당 유저의 일반 카드팩 잔여 수량이 1 증가한다.
- `events`에 `ticket.granted` 로그가 남아 나중에 추적할 수 있다.
- 지급 후 앱 가입자 목록과 보상 현황이 갱신된다.

## 범위

- 카드 뽑기권만 지급한다.
- 추첨권, 카드 직접 지급, 조건 기반 누락 보상 정책은 건드리지 않는다.
- 이름/닉네임 검색은 기존 앱 가입자 검색 UI를 그대로 사용한다.
