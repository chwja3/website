# Admin Dashboard Layout Tuning Plan

## 목표

admin 대시보드에서 교구별 이번 주 참여 영역을 주차별 항목 기록보다 먼저 보이게 하고, 교구 카드가 앱 가입자 명단의 교구 badge 색상과 같은 계열로 보이게 한다.

## 구현 방향

1. `교구별 이번 주 참여` 카드와 `주차별 항목 기록` 카드의 DOM 순서를 바꾼다.
2. 교구 요약 grid를 데스크톱 4열로 고정하고, 화면이 좁아지면 2열 또는 1열로 줄인다.
3. 교구별 summary key에 맞는 색상 class를 부여한다.
4. `날짜 합계` 라벨은 줄바꿈되지 않도록 처리한다.

## 검증 기준

- 대시보드에서 교구별 참여가 주차별 항목 기록보다 위에 나온다.
- 1교구, 2교구, 3교구, 4교구 카드는 앱 가입자 badge와 같은 색상 계열이다.
- 데스크톱 폭에서 한 줄에 교구 카드가 4개까지 배치된다.
- `날짜 합계` 텍스트가 `날짜합/계`처럼 쪼개지지 않는다.
