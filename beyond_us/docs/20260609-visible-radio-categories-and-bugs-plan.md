# 2026-06-09 보이는 라디오 카테고리와 운영 버그 수정 계획

## 목표
- 보이는 라디오 사연 입력을 기획안 기준 6개 카테고리로 분류한다.
- admin에서 진행자가 실시간으로 카테고리, 대상, 시간순으로 사연을 확인할 수 있게 한다.
- 교환 화면에서 기도제목이 보이지 않는 문제를 수정한다.
- admin 실물 카드 수령 탭에서 네헤와 더무 카드명이 뒤바뀌는 문제를 수정한다.

## 구현 방향
- Supabase `visible_radio_stories`에 `category_key`, `category_label`, `target_text`, `status`, `pinned_order`를 추가한다.
- 사용자 입력 UI는 카테고리 선택, 대상자 선택 입력, 사연 작성 순서로 바꾼다.
- admin 목록은 카테고리 칩, 대상, 시간, 상태를 표시하고 카테고리와 검색어로 필터링한다.
- 교환 기도제목은 `get_user_trades` RPC와 사용자 렌더링 중 누락 지점을 확인해 보강한다.
- 실물 카드 수령 카드명은 admin 카드 메타와 DB 카드 id 매핑을 같은 기준으로 맞춘다.
