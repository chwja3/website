# 미션 탭 노출 토글 Context

## 배경

사용자에게 너무 많은 현장 기능이 한 번에 보이면 앱 진입 초기에 혼란이 생길 수 있다는 의견이 있었다. 반대로 목회자 관점에서는 Coming Soon으로라도 앞으로 열릴 기능이 보이는 편을 선호할 수 있다. 그래서 코드로 고정하지 않고 admin 토글로 Q.T., B.B.B, 천로역정 탭 노출을 조정할 수 있게 한다.

## 현재 구조

- `TabSettings`는 `tab_key`, `label`, `enabled` 구조다.
- 앱 dashboard 응답의 `tabSettings`를 `applyTabSettings()`가 읽어서 드로어 메뉴를 숨기거나 보인다.
- Q.T.는 이미 `qt` 키가 있다.
- B.B.B 미션 탭은 기존 `secret` 키를 사용한다.
- 천로역정은 앱 섹션이 분리되어 있지만 탭 토글 키가 없었다.

## 결정

- 천로역정 탭 키는 `pilgrim`으로 둔다.
- 천로역정은 새 키가 없거나 `true`가 아니면 숨긴다.
- Q.T.와 B.B.B는 기존 시트 값이 `false`일 때 숨기는 기존 호환성을 유지한다.
- B.B.B 내부 M1, M2, M3 오픈 상태는 기존 `BBBSettings` 섹션 토글과 분리한다.
