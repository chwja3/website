# 숙소 배정 전체 PDF 게시 준비 계획

## 목표

숙소 배정 PDF 하나를 모든 유저가 같은 숙소/차량 탭에서 볼 수 있게 준비한다.

## 범위

- Supabase Storage의 고정 경로 `logistics/lodging_assignment.pdf`를 앱에서 확인한다.
- PDF가 있으면 전체 배정표 PDF 미리보기와 열기 버튼을 보여준다.
- PDF가 없으면 기존 개인별 숙소/차량 배정 정보 화면을 유지한다.
- 관리자 배정표 데이터와 매칭 RPC는 건드리지 않는다.

## 검증

- 앱 스크립트 문법 검사를 통과한다.
- 앱 HTML 인라인 스크립트 검사를 통과한다.
- 앱 버전과 서비스워커 캐시 버전을 함께 갱신한다.
