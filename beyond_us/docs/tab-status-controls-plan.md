# 탭 상태 표시 관리 계획

## 목표

admin의 `탭 활성화` 화면에서 유저 앱의 모든 탭을 관리한다. 각 탭은 앱에서 보일지 여부와 현재 사용 가능 상태를 분리해서 관리한다.

## 방향

1. `TabSettings`에 `status` 열을 추가한다.
2. `enabled`는 앱 메뉴 노출 여부로 유지한다.
3. `status=open`은 유저 앱 메뉴에 초록색 표시, `status=closed`는 빨간색 표시로 렌더링한다.
4. 기존 `prayer`, `secret`, `qt`, `pilgrim`, `specialPack` API 호환성은 유지한다.
5. admin 화면은 GAS가 내려주는 탭 목록을 기준으로 동적으로 렌더링한다.

## 검증 기준

- 기존 3열 `TabSettings` 시트도 자동으로 4열 구조로 보정된다.
- admin에서 모든 유저 앱 탭의 표시 여부와 초록/빨강 상태를 저장할 수 있다.
- 유저 앱 드로어에서 숨김 탭은 보이지 않고, 표시 탭은 초록/빨강 점이 보인다.
