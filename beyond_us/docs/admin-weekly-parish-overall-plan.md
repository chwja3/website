# 주차별 교구 전체 요약 계획

admin 대시보드의 주차별 교구 참여 기록에서 교구별 카드 뒤에 전체 합계 카드를 함께 보여준다.

1. 기존 Supabase `admin_dashboard_summary`가 내려주는 주차별 교구 summary를 그대로 사용한다.
2. 프론트에서 선택 주차의 교구 summary를 합산해 전체 참여자, 날짜 합계, 점수를 계산한다.
3. 전체 카드는 `교회학교/목양교구` 바로 뒤에 표시한다.
4. 전체 카드를 클릭하면 해당 주차 전체 참여자 상세 목록을 확인할 수 있게 한다.

검증 기준은 주차별 교구 참여 기록에서 `전체` 카드가 보이고, 카드의 참여자/날짜 합계/점수가 각 교구 카드 합계와 일치하는 것이다.
