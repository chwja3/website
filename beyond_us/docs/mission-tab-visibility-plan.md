# 미션 탭 노출 토글 Plan

## 목표

Q.T. 말씀 묵상, B.B.B 미션, 천로역정 탭을 admin의 탭 활성화 화면에서 각각 켜고 끌 수 있게 한다. 운영 정책이 확정되기 전에는 탭 자체를 숨길 수 있어야 하고, 필요하면 Coming Soon 콘텐츠를 노출하는 방식으로 다시 켤 수 있어야 한다.

## 작업 범위

- 기존 Q.T. 토글은 유지한다.
- 기존 B.B.B 미션 토글은 유지한다.
- 천로역정 전용 `pilgrim` 탭 토글을 추가한다.
- 앱 드로어에서 비활성화된 탭은 숨긴다.
- 현재 보고 있는 탭이 비활성화되면 사전미션으로 돌려보낸다.

## 제외 범위

- B.B.B 내부 섹션별 Coming Soon 문구와 오픈 토글은 기존 B.B.B 미션 admin 화면에서 계속 관리한다.
- 실제 운영 날짜별 자동 오픈 예약은 이번 작업에 포함하지 않는다.
