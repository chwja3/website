# BBB 사진 없는 수동 승인 계획

사진 업로드가 실패한 사용자도 운영진이 BBB Mission 1, Mission 2를 성공 처리할 수 있게 만든다.

## 확인 기준

- BBB/천로역정 운영 현황에서 사진이 없어도 Mission 1, Mission 2 성공 처리가 가능하다.
- 기존 사진 제출 승인과 거절 흐름은 그대로 유지된다.
- 수동 성공 처리도 기존 승인 보상과 같은 카드팩 지급 경로를 사용한다.
- 사용자 앱에는 수동 승인 marker가 깨진 이미지로 보이지 않는다.
- 앱 변경이 포함되므로 캐시 버전을 함께 갱신한다.

## 작업 순서

1. admin BBB 운영 UI와 사진 승인 RPC 흐름을 확인한다.
2. 사진 제출 row 없이도 Mission 1, Mission 2를 성공 처리하는 RPC를 추가한다.
3. admin UI에서 사진 없는 미션에도 수동 성공 버튼을 연결한다.
4. 사용자 앱에서 수동 승인 상태가 깨진 이미지 URL로 보이지 않게 한다.
5. 안성재에게 적용할 SQL과 admin 절차를 남긴다.
6. 문법과 diff 검사를 실행한다.
