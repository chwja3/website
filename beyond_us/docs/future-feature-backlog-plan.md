# 향후 기능 백로그 Plan

## 목적

수련회 현장 운영 전까지 남은 기능 아이디어를 한곳에 모아, 구현 가능한 단위와 기획 확인이 필요한 단위를 분리한다.

## 현재 완료로 분류한 항목

- 현장미션 MISSION 3 위치 정리. 사용자 기준 완료 항목으로 기록한다.
- Q.T. 말씀 묵상 페이지 준비중 화면.
- 구성 탭 레이아웃.

## 남은 큰 작업 묶음

1. BBB 조별 매칭 정책 재설계.
2. 히든 카드 미션 수정.
3. 크라임씬 맵 기반 광범위수사 기능.
4. 내 안의 빛 유형검사 문항과 결과 로직.
5. 레어카드 뽑기와 카드팩 이펙트 강화.
6. 개인별 현장 미션 2개 배정.
7. 추첨권 운영 고도화.

## 진행 원칙

- 의미가 불분명한 항목은 바로 구현하지 않는다.
- 각 기능은 먼저 데이터 구조, 보상 정책, admin 운영 방식, 앱 UX를 확정한다.
- 현장 운영자가 확인해야 하는 기능은 admin 페이지 작업을 함께 설계한다.

## 2026-05-13 반영

- 추첨권 시스템 1차 구현은 컬렉션 보유 종류 기준 표시와 추첨통 모달까지 완료했다.
- 참석 추첨권 1장은 admin 수련회 참석 탭으로 반영했다.
- 추첨권 구현은 여기서 1차 마무리로 본다.
- 확인용 DEV 프리뷰는 유지하며, 남은 범위는 추첨권 UI 디테일 조정과 admin 당첨자 추첨/수동 조정 화면이다.
