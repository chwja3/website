# B.B.B. 이름 표시 정리 계획

## 목표

B.B.B. 사진 승인, B.B.B./천로역정 운영 현황, B.B.B. 매칭 화면에서 사람이 닉네임이나 `null`로 보이지 않도록 이름을 우선 표시한다.

## 범위

- 관리자 UI의 표시 문자열만 수정한다.
- 검색과 저장에 필요한 내부 식별자는 유지한다.
- Supabase RPC와 데이터 구조는 변경하지 않는다.

## 검증

- `admin.html` 스크립트 문법 검사를 통과한다.
- B.B.B. 관련 표시 함수에서 문자열 `null`이 그대로 노출되지 않도록 확인한다.
