- 미션 완료 보상은 카드 뽑기권, 즉 일반 카드팩 1장이다.

- 보상 조건 날짜는 `2026-06-20` 하나로 제한한다.

- 보상 중복 방지는 `events`의 `ticket.granted`, `ref_type = 'qt'`, `ref_id = '2026-06-20'`, `payload.reason = 'qt_2026_06_20_complete'` 조합으로 확인한다.
- `2026-06-21` QT는 같은 입력 UI를 쓰지만 보상은 지급하지 않는다.

- 6월 20일에 일부만 저장한 뒤 나중에 네 칸을 모두 채우면 그 저장 시점에 보상이 지급된다.
