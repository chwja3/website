# Backup Modern Schema Checklist

- [x] 백업 파일명에 현재 스키마 버전을 넣는다.
- [x] 백업 사본에 현재 필수 시트와 헤더를 보장한다.
- [x] 백업 사본의 탭 순서, 색상, 숨김 상태를 현재 운영 구조로 맞춘다.
- [x] 백업 결과 JSON에 정규화 결과를 포함한다.
- [x] 백업 복사 성공과 정규화 실패를 분리해 백업 생성 자체가 실패로 보이지 않게 한다.
- [x] 오래된 백업 정리는 파일명 대신 생성일 기준으로 처리한다.
- [x] 로컬 문법 검증을 실행한다.
- [x] 커밋과 푸시를 완료한다.
