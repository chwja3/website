# 2026-06-12 관리자 카드 뽑기권 미지급 기준 지급 컨텍스트

## 배경

- 사용자가 카드 뽑기권 누락 보상은 조건 미충족 여부와 상관없이, 지급된 적이 없으면 지급 가능하게 해달라고 요청했다.

## 결정

- 카드 뽑기권 누락 보상의 지급 가능 여부는 `already_claimed`만 본다.
- `conditionMet` 값은 계속 내려주지만 버튼 활성화 기준에는 쓰지 않는다.
- 사전미션 보상은 기존과 같은 `mission_week_threshold` 이벤트 reason을 남겨 중복 지급을 막는다.
- H&P 보상은 기존과 같은 `ref_type = hold_pray`, `week_key = w3/w6` 이벤트를 남겨 중복 지급을 막는다.

## 주의점

- 이 기능은 관리자 수동 보정 기능이다.
- 실제 유저 앱의 자동 지급 정책과는 별도다.
