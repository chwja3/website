# 2026-06-13 QT 완료 보상 계획

## 목표

2026년 6월 20일 QT에서 답변 3개와 기도제목을 모두 작성하면 미션 완료로 처리하고 일반 카드팩 1장을 지급한다.

## 성공 기준

- 2026년 6월 20일 QT에서 답변 1, 답변 2, 답변 3, 기도제목이 모두 입력되면 카드팩 1장이 지급된다.
- 같은 사용자가 같은 QT 보상을 여러 번 받을 수 없다.
- 2026년 6월 21일 QT는 답변 저장만 하고 카드팩 보상은 지급하지 않는다.
- 보상 지급 시 앱에 미션 완료 메시지가 표시되고 카드팩 배지가 갱신된다.

## 구현 범위

- Supabase `submit_qt_reflection_v2` RPC에 6월 20일 완료 보상 조건을 추가한다.
- 보상은 기존 카드팩 지급 방식처럼 `user_inventory`와 `events`에 반영한다.
- 앱 저장 완료 메시지를 보상 결과에 맞춰 분기한다.
