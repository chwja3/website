# 2026-06-18 조별 명단과 앱 가입자 매칭 계획

## 목표

`조별 데이터 정리 6_14.xlsx`의 조 편성 명단과 PROD 앱 가입자 명단을 이름과 교구 기준으로 매칭한다.

## 산출물

1. 조별 매칭 결과.
2. 조에는 있으나 앱 닉네임이 없는 사람.
3. 앱에는 가입했으나 조 편성 명단에 없는 사람.
4. 이름 중복으로 확인이 필요한 사람.
5. admin `B.B.B. 미션` 탭에서 동일한 조 편성 기준으로 보이는 운영 화면.

## 기준

- 조에 있는 사람은 조 명단에 남겨두고, 앱 가입자가 없으면 `닉네임 없음`으로 표시한다.
- 이름 중복자는 교구까지 비교해 자동 매칭을 시도한다.
- 이름과 교구가 모두 중복되면 `이름 중복 확인필요`로 별도 표기한다.
- 앱 가입자는 있으나 조 명단에 없으면 `미참` 탭으로 모은다.
- DB에는 원본 조 명단을 별도 테이블로 보관하고, 확실히 매칭된 앱 가입자만 `group_members`에 연결한다.
- admin BBB 매칭 화면은 원본 조 명단 기준으로 렌더링한다.
