# BBB 사진 촬영 우선 열기 컨텍스트

2026-06-19. 사용자는 BBB 사진 업로드가 사진첩만 열리는 것처럼 보인다고 했다. 앱에서 실제 노출되는 파일 input은 `bbbPhotoInput`, `bbbM2Input` 두 개이며, 천로역정은 QR 인증 방식이라 사진 input이 정상 경로에서는 쓰이지 않는다.

모바일 브라우저가 카메라 앱을 우선 열도록 두 input에 `capture="environment"`를 추가했다. 브라우저와 OS 정책에 따라 사용자가 카메라와 사진첩 선택지를 볼 수도 있지만, 가능한 범위에서 카메라 촬영을 우선 유도하는 표준 속성이다.
