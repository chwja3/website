# Raffle Available Numbers Plan

## 목표

admin 추첨권 번호 탭에서 회수된 번호 개수뿐 아니라 어떤 번호가 회수되어 재사용 가능한지도 함께 보여준다.

## 구현 방향

1. `adminGetRaffleTickets` 응답에 비활성 추첨권 번호 목록을 추가한다.
2. admin 통계 카드의 `회수된 번호` 아래에 번호 chip 목록을 렌더링한다.
3. 목록이 길어질 경우 일부만 표시하고 나머지 개수를 안내한다.

## 검증 기준

- 회수된 번호가 없으면 `없음`으로 보인다.
- 회수된 번호가 있으면 `#0007` 같은 형식으로 표시된다.
- 기존 추첨권 번호 테이블과 검색 기능은 그대로 동작한다.
