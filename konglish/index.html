사용 방법

1

한국어 입력

왼쪽 입력창에 변환하고 싶은 한국어 텍스트를 입력하세요. 최대 5,000자까지 지원합니다.

2

자동 변환

입력하는 즉시 오른쪽에 콩글리쉬 변환 결과가 실시간으로 표시됩니다.

3

클립보드 복사

오른쪽 상단의 복사 버튼을 누르면 변환 결과가 클립보드에 복사됩니다.

4

어디서나 붙여넣기

SNS, 메신저, 문서 등 원하는 곳에 Ctrl+V (또는 길게 눌러 붙여넣기)로 사용하세요.

자주 묻는 질문 (FAQ)

콩글리쉬는 한국어(Korean)와 영어(English)의 합성어로, 한국어 발음을 영문 알파벳으로 표기하는 방식입니다. K-팝, K-드라마 팬들 사이에서 한국어 단어를 영어로 표현할 때 주로 사용됩니다. 예를 들어 "오빠(oppa)", "사랑해(saranghae)" 같은 표현이 세계적으로 유명해졌습니다.
국립국어원의 로마자 표기법을 기반으로 하되, 실제 K-팝·K-드라마·SNS에서 널리 쓰이는 콩글리쉬 스타일로 조정했습니다. 학술적 표기보다는 실용적이고 친숙한 표기를 우선합니다.
네, 완전 무료입니다. 회원가입이나 로그인 없이 바로 사용할 수 있습니다.
아니요. 모든 변환은 브라우저 내에서만 처리되며, 입력한 텍스트는 서버로 전송되거나 저장되지 않습니다. 개인 정보나 민감한 내용도 안심하고 사용하실 수 있습니다.
한글 외의 문자(영문, 숫자, 특수문자, 공백)는 변환 없이 그대로 출력됩니다. 한글과 영문이 섞인 텍스트도 자연스럽게 변환됩니다.
네, 스마트폰과 태블릿에서도 최적화되어 사용할 수 있습니다. 모바일에서는 상하 레이아웃으로 자동 전환됩니다.
